Streetlight Automation System

Streetlight Automation System is an IOT based solution to reduce manual intervention in operating streetlights and provide better notification in case of minor faults. The system comprises a hardware system using a node mcu board, attached to streetlights and an accompanying mobile application which can be used for monitoring and notifications.
