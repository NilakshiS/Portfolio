# Bootstrap4Website
This is a simple webpage built using the boilerplate v7.2.0 template and bootstrap v4.3. It has the following features:
- Gradient text background on hover on links
- Carousel
- Responsive Cards
- Parallax Effect on a full size image
- Modal
- Icons
- Footer
